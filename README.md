## 1、项目介绍

基于SSM的图书管理管理系统4拥有两种角色，用户可以浏览评论图书、登录注册，管理员可以进行图书馆管理、用户管理、分类管理等功能


## 2、项目技术

后端框架：SSM（Spring、SpringMVC、Mybatis）

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql5.x
- maven版本：不需要
- 硬件环境：Windows 或者 Mac OS
